========================================
ПРОСТО ПРО URL API
========================================

URL API - ЭТО АДРЕС ТВОЕЙ МОДЕЛИ
----------------------------------

ГДЕ ЕГО ВЗЯТЬ:

1. ЕСЛИ У ТЕБЯ OLLAMA:
   - Запусти: ollama serve
   - URL будет: http://localhost:11434
   - ВВЕДИ ЭТО В НАСТРОЙКАХ

2. ЕСЛИ У ТЕБЯ LM STUDIO:
   - Запусти LM Studio
   - Включи Local Server
   - URL будет: http://localhost:1234
   - ВВЕДИ ЭТО В НАСТРОЙКАХ

3. ЕСЛИ ДРУГОЙ СЕРВЕР:
   - Посмотри на каком порту он работает
   - URL: http://localhost:ПОРТ
   - Например: http://localhost:8000

4. ЕСЛИ НЕ ЗНАЕШЬ:
   - Попробуй: http://localhost:11434
   - Или: http://localhost:1234
   - Или: http://localhost:8000

КАК ПРОВЕРИТЬ:
--------------
1. Открой браузер
2. Вставь URL (например: http://localhost:11434)
3. Если видишь ответ - работает!
4. Если ошибка - сервер не запущен

========================================
ГДЕ ВВЕСТИ В РАСШИРЕНИИ:
========================================

1. Открой расширение (кликни на иконку)
2. Нажми на шестеренку (настройки)
3. Введи URL (например: http://localhost:11434)
4. Готово!
