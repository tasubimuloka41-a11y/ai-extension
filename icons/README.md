# Иконки расширения

Поместите сюда следующие файлы иконок:

- `icon16.png` - 16x16 пикселей
- `icon48.png` - 48x48 пикселей  
- `icon128.png` - 128x128 пикселей

Вы можете создать иконки самостоятельно или использовать онлайн генераторы иконок для расширений браузера.

Рекомендуется использовать иконку с символом AI или буквой "G" для Gemma.
