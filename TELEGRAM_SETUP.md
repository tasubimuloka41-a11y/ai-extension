# Настройка Telegram бота

## Как получить токен бота

1. Открой Telegram
2. Найди бота **@BotFather**
3. Отправь команду: `/newbot`
4. Следуй инструкциям:
   - Введи имя бота (например: "Мой AI Ассистент")
   - Введи username бота (должен заканчиваться на `bot`, например: `my_ai_assistant_bot`)
5. BotFather даст тебе токен (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
6. Скопируй этот токен

## Настройка в расширении

1. Открой расширение
2. Нажми на иконку Telegram (в заголовке)
3. Вставь токен в поле "Токен бота"
4. Нажми "Сохранить"
5. Нажми "▶ Запустить"

## Как это работает

- Бот получает сообщения в Telegram
- Передает их твоей модели AI (Gemma 3 12B)
- Модель генерирует ответ
- Бот отправляет ответ обратно в Telegram

## Пример использования

1. Найди своего бота в Telegram (по username, который ты указал)
2. Начни диалог с ботом
3. Отправь сообщение (например: "Привет!")
4. Бот ответит через твою модель AI

## Остановка бота

- Нажми "⏹ Остановить" в панели Telegram
- Или закрой расширение

## Безопасность

- Токен хранится локально в расширении
- Никто кроме тебя не имеет доступа
- Все сообщения обрабатываются локально через твою модель
